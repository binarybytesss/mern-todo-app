🌈 Kids Colorful MERN To-Do App ✨

A fun and colorful To-Do List application designed especially for kids 🎨💖.
This app turns daily tasks like homework 📚 and chores 🧹 into an exciting and happy experience ⭐🎉.

Built using the MERN Stack (MongoDB, Express, React, Node.js).

🚀 Features

➕ Add new tasks

✅ Mark tasks as completed

⭐ Delete tasks

🎨 Attractive and kid-friendly UI

🔄 Real-time updates

🛠️ Tech Stack

Frontend:

React.js ⚛️

Axios

Backend:

Node.js

Express.js

Database:

MongoDB 🍃

📂 Project Structure
To-do-app
│
├── client   (React Frontend)
└── server   (Node + Express Backend)
⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/binarybytesss/mern-todo-app.git
cd mern-todo-app
2️⃣ Setup Backend
cd server
npm install
npm start

Backend runs on:

http://localhost:5000
3️⃣ Setup Frontend

Open a new terminal:

cd client
npm install
npm start

Frontend runs on:

http://localhost:3000
🌍 Deployment 

Frontend → Vercel

Backend → Render

Database → MongoDB Atlas

🎯 Purpose

This project helps children build responsibility, good habits, and time management skills in a fun and interactive way 🌟🎈.
